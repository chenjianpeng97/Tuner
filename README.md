- .github/skills下记录了一些目前vibecoing用到的一些skills，如.github/skills/http-stage-behave-testing/，onboard时应选择性查看

- notes/README.md和spec.md是一些项目相关的笔记和规范说明，onboard时也建议查看
- `.specify/memory/constitution.md` 是工程流程与质量门禁的最高优先级规范，规划/实现/评审时需对齐
